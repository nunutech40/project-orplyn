# Manifest Canonical Aset Terkurasi

## Aturan membaca

1. Mulai dari lane sesuai kebutuhan, bukan dari nama file lama.
2. `production-orders/` memiliki konteks order yang lebih kuat; tetap ikuti batas klaim per file.
3. `finished/` menunjukkan contoh teknik/hasil, tetapi tidak otomatis terhubung dengan customer, event, jumlah, deadline, atau kepuasan.
4. `process/` hanya menunjukkan aktivitas proses yang terlihat. Jangan menghubungkan proses dengan order tertentu tanpa konfirmasi.
5. Bukti publik tetap mengikuti `assets/proof/`, `marketing/05-content/proof-asset-audit-2026-07-20.md`, dan Public-Claim Gate.
