# Foto Produksi Orplyn

Folder ini adalah arsip sumber, bukan folder publik website. Nama file dibuat ASCII, stabil, dan singkat agar mudah dicari oleh manusia maupun AI.

## Struktur

- `orders/`: foto hasil order yang memiliki konteks dari owner.
- `process/`: foto proses produksi. Foto ini membuktikan aktivitas proses, tetapi tidak otomatis membuktikan hasil order, pelanggan, jumlah, atau ketepatan waktu tertentu.
- `ASSET-MANIFEST.md`: metadata, konteks, dan batas klaim untuk setiap aset.

## Aturan pakai

1. Gunakan manifest sebelum memilih aset untuk website, Ads, GBP, atau konten.
2. Jangan menyimpulkan semua order di `orders/` adalah sablon manual. Empat order yang konteksnya tersedia saat ini memakai DTF atau kombinasi DTF; DTF adalah teknik digital.
3. Jangan menyimpulkan foto `process/` terkait sebuah acara/customer tertentu tanpa konteks tambahan dari owner.
4. Owner memberi izin penggunaan aset folder ini untuk website dan iklan pada 20 Juli 2026. Public-Claim Gate tetap berlaku: pilih hanya fakta yang didukung aset/metadata dan ikuti batas klaim di manifest.
5. Screenshot chat tanpa nama acara tetap dapat menjadi feedback pelanggan, tetapi bukan bukti bahwa order tersebut event/komunitas kecuali konteksnya terlihat atau dikonfirmasi owner.
